# Discovery Questions

## Q1: Will users interact with this face swapper through a web-based interface?
**Default if unknown:** Yes (web-based applications are most accessible and require no installation)

## Q2: Should the application support real-time face swapping (via webcam/video)?
**Default if unknown:** No (static image processing is simpler to start with)

## Q3: Will this application need to handle multiple faces in a single image?
**Default if unknown:** Yes (group photos are a common use case)

## Q4: Should users be able to save and download their face-swapped results?
**Default if unknown:** Yes (users typically want to keep their creations)

## Q5: Will this require user authentication and personal storage?
**Default if unknown:** No (allows for simpler, privacy-focused anonymous usage)
